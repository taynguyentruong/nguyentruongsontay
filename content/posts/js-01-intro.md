JavaScript (JS) là ngôn ngữ kịch bản cho phép bạn triển khai các tính năng phức tạp trên trang web.

## Biến trong JS
Sử dụng `let`, `const`, hoặc ``var` (hạn chế dùng var).

```javascript
let message = "Xin chào JavaScript";
const PI = 3.14;

console.log(message);
```

JavaScript có thể chạy trên trình duyệt (client-side) hoặc server (Node.js).
