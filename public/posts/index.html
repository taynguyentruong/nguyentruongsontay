Bài 9: Bất đồng bộ trong JavaScript (Async/Await)

JavaScript là ngôn ngữ đơn luồng, nhưng xử lý các tác vụ IO (như gọi API) một cách bất đồng bộ. Promise Là một đối tượng đại diện cho một tác vụ chưa hoàn thành. Async / Await Cú pháp “ngọt” (syntactic sugar) giúp viết code bất đồng bộ trông giống đồng bộ. async function fetchData() { try { const response = await fetch("https://api.example.com/data"); const data = await response.json(); console.log(data); } catch (error) { console.error("Lỗi:", error); } } fetchData(); Đây là cách hiện đại và dễ đọc nhất để xử lý Ajax.

Stream API được giới thiệu trong Java 8 giúp xử lý các collection theo phong cách functional. Bài 2: Các khái niệm cơ bản của OOP trong Java

Lập trình hướng đối tượng (OOP) là nền tảng của Java. Có 4 tính chất chính: Tính đóng gói (Encapsulation): Che giấu dữ liệu bên trong class. Tính kế thừa (Inheritance): Class con kế thừa thuộc tính và phương thức của class cha. Tính đa hình (Polymorphism): Một phương thức có thể có nhiều cách thực hiện khác nhau. Tính trừu tượng (Abstraction): Ẩn đi chi tiết triển khai, chỉ hiển thị tính năng. class Animal { void makeSound() { System.out.println("Some sound..."); } } class Dog extends Animal { @Override void makeSound() { System.out.println("Gâu gâu"); } }

Bài 1: Làm quen với Java

Java là một ngôn ngữ lập trình hướng đối tượng, đa mục đích và rất phổ biến. Điểm mạnh của Java là “Viết một lần, chạy mọi nơi” (Write Once, Run Anywhere). Cài đặt JDK Để lập trình Java, bạn cần cài đặt JDK (Java Development Kit). Bạn có thể tải JDK từ trang chủ Oracle hoặc OpenJDK. Chương trình Hello World public class HelloWorld { public static void main(String[] args) { System.out.println("Xin chào Java!"); } } Khi chạy chương trình này, màn hình sẽ in ra dòng chữ “Xin chào Java!”.
