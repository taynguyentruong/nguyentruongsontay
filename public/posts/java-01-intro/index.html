Bài 1: Làm quen với Java

Giới thiệu về ngôn ngữ lập trình Java và cách cài đặt môi trường.
December 20, 2024 · Nguyễn Trường Sơn Tây

Java là một ngôn ngữ lập trình hướng đối tượng, đa mục đích và rất phổ biến. Điểm mạnh của Java là “Viết một lần, chạy mọi nơi” (Write Once, Run Anywhere).

Cài đặt JDK#

Để lập trình Java, bạn cần cài đặt JDK (Java Development Kit). Bạn có thể tải JDK từ trang chủ Oracle hoặc OpenJDK.

Chương trình Hello World#

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào Java!");
    }
}

Khi chạy chương trình này, màn hình sẽ in ra dòng chữ “Xin chào Java!”.
