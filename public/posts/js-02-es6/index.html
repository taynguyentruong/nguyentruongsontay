Bài 7: Các tính năng mới trong ES6

Arrow function, Template literal và Destructuring.
December 23, 2024 · Nguyễn Trường Sơn Tây

ES6 (ECMAScript 2015) mang lại nhiều cải tiến lớn cho JavaScript.

Arrow Function#

Cú pháp ngắn gọn hơn cho hàm.

const sum = (a, b) => a + b;

Template Literals#

Sử dụng backticks ` để nội suy chuỗi.

let name = "Sơn Tây";
console.log(`Xin chào ${name}`);

Destructuring#

Giải nén giá trị từ mảng hoặc đối tượng.

const user = { id: 1, name: "Admin" };
const { name } = user;
