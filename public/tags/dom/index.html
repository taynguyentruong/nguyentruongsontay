Bài 8: DOM Manipulation trong JavaScript

DOM (Document Object Model) biểu diễn trang web dưới dạng cây các đối tượng. Truy xuất phần tử const btn = document.getElementById("myBtn"); const items = document.querySelectorAll(".item"); Thay đổi nội dung btn.innerText = "Đã click!"; btn.style.backgroundColor = "blue"; Sự kiện (Events) btn.addEventListener("click", () => { alert("Button clicked!"); });
