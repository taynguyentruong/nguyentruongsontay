Bài 3: Sử dụng Java Collections Framework

Java Collections Framework cung cấp các cấu trúc dữ liệu mạnh mẽ. List List lưu trữ các phần tử theo thứ tự và có thể lặp lại. List<String> list = new ArrayList<>(); list.add("Java"); list.add("Python"); Set Set lưu trữ các phần tử không trùng lặp. Set<String> set = new HashSet<>(); set.add("A"); set.add("A"); // Bỏ qua Map Map lưu trữ dữ liệu dưới dạng Key-Value. Map<String, Integer> map = new HashMap<>(); map.put("One", 1);
